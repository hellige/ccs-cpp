---

Tied specificities
---

---

Complex tie
---
bar.class2 : test1 = "incorrect"
bar.class1 : test1 = "correct"

bar foo : test2 = "incorrect 2"

===

Conjunction specificities
---

---

Context
---

Trailing combinator
---
a: test = "top"
a { b >: test = "bottom" }

test2 = "top"
a > { b { c > { d { e: test2 = "bottom" }}}}
---
b a: test = bottom
b a e c b d: test2 = bottom
b a e c d: test2 = top
===

Order-dependent disjunction
---
a, b, d { x = 1 }

c { x = 3 }

(d, e) > f { y = 1 }
d > f { y = 2 }
---
b: x = 2
a c d: x = 3
d f: y = 2
===

---

---
